Hello, welcome to Foodel

Foodel is a simple program that was originally designed as an example to help teach students about practical uses of artificial intelligence. It was realised that during the COVID-19 lockdown Foodel could be used to help businesses and organisations make efficient home deliveries. Foodel is offered free of charge to anyone or any organisation who can make use of it during the COVID-19 crisis. Foodel takes its input in a simple spreadsheet and produces GPX and KML files which may be uploaded to maps and GPS devices. Foodel is written in Java and can be run on Windows as well as Mac OS. It uses Open Streetmap data and GraphHopper for routing, the rounds are organised using a state of the art Evolutionary Algorithm. It is designed to be simple, friendly and easy to use.
