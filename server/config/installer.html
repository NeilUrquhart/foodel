Installer

This page will guide you through the process of creating an installer for Foodle which you can use on your own machine(s). The details entered on this form are not stored on our servers; they are used only to tailor your install.

Operating System

What type of operating system (OS) will you be running Foodel on?



Why is this information required?

If you choose Windows, you will recieve an .exe file which works natively on Windows. Mac users will recieve a JAR file which can be used on Mac's with Java installed. Other operating systems should work too, but this has not been tested.

Organisation Details

Why is this information required?

Your organisation's name will appear in the footer of your site.

Map Area





Why is this information required?

We need to know the rough location of where Foodel will be used to provide you with accurate routing information.

You can pick the relevant map area to your location. The UK option includes Scotland, England, and Wales but incurs a larger file size.

Device Install


What is the difference?

A local installation is on your own machine and is most likely not accessible from the internet. In this install, you don't need to keep track of your job key to access your job. On a shared install, one that is likely accessible from the internet, the job key is required.
